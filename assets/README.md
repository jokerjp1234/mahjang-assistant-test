# アセットディレクトリ

このディレクトリには、麻雀アシスタントで使用する画像ファイルなどのリソースを格納します。

## ディレクトリ構成

- `tiles/` - 麻雀牌の画像
  - `m1.png` - 1萬
  - `m2.png` - 2萬
  - ...
  - `p1.png` - 1筒
  - ...
  - `s1.png` - 1索
  - ...
  - `zeast.png` - 東
  - `zsouth.png` - 南
  - `zwest.png` - 西
  - `znorth.png` - 北
  - `zwhite.png` - 白
  - `zgreen.png` - 發
  - `zred.png` - 中

## 注意事項

- 牌画像は40x50ピクセルが推奨サイズです
- PNG形式で透過背景のものを使用してください
